# 📱 전체 화면 리스트 및 구현 현황

## 🎯 화면 구조 개요

### 사용자 유형별 분류
1. **공통 화면** - 모든 사용자
2. **인플루언서 화면** - 일반 유저 (체험단 참여자)
3. **점주 웹 화면** - 점주 관리 페이지 (데스크톱/태블릿)
4. **점주 앱 화면** - 점주 및 직원용 모바일 앱
5. **슈퍼 관리자 화면** - 전체 시스템 관리
