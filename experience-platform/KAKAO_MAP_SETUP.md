# 카카오 지도 API 설정 가이드

## 🎯 필요한 작업

네이버 지도 API 인증 문제로 인해 카카오 지도 API로 교체했습니다.
아래 단계를 따라 카카오 지도 API 키를 발급받고 설정해주세요.

---

## 📋 1단계: 카카오 Developers 계정 생성 및 앱 등록

### 1-1. 카카오 Developers 접속
- 사이트: https://developers.kakao.com
- 카카오 계정으로 로그인

### 1-2. 애플리케이션 추가하기
1. 상단 메뉴에서 **"내 애플리케이션"** 클릭
2. **"애플리케이션 추가하기"** 버튼 클릭
3. 앱 정보 입력:
   - **앱 이름**: `체험단 플랫폼` (또는 원하는 이름)
   - **사업자명**: 개인 또는 회사명
4. **저장** 클릭

---

## 🔑 2단계: JavaScript 키 발급

### 2-1. 앱 키 확인
1. 생성한 앱을 클릭
2. 좌측 메뉴에서 **"앱 키"** 선택
3. **"JavaScript 키"** 복사

```
예시: 1234567890abcdef1234567890abcdef
```

### 2-2. .env 파일에 키 등록

프로젝트 루트의 `.env` 파일을 열고 다음과 같이 수정:

```bash
# 카카오 지도 API
VITE_KAKAO_MAP_APP_KEY=여기에_발급받은_JavaScript_키_입력
```

**예시:**
```bash
VITE_KAKAO_MAP_APP_KEY=1234567890abcdef1234567890abcdef
```

---

## 🌐 3단계: 플랫폼 등록 (중요!)

### 3-1. Web 플랫폼 추가
1. 좌측 메뉴에서 **"플랫폼"** 선택
2. **"Web 플랫폼 추가"** 버튼 클릭
3. **사이트 도메인** 입력:
   ```
   http://localhost:5173
   ```
4. **저장** 클릭

### 3-2. 추가 도메인 등록 (배포 시)
배포 후 실제 도메인도 추가해야 합니다:
```
https://your-domain.com
```

---

## ✅ 4단계: 설정 확인

### 4-1. 개발 서버 재시작
```bash
# 기존 서버가 실행 중이면 Ctrl+C로 종료 후
npm run dev
```

### 4-2. 브라우저에서 확인
```
http://localhost:5173
```

지도가 정상적으로 표시되면 설정 완료!

### 4-3. 테스트 페이지 확인 (선택사항)
```
http://localhost:5173/test-map.html
```

---

## ⚠️ 주의사항

### 1. JavaScript 키 노출 주의
- `.env` 파일은 절대 GitHub에 커밋하지 마세요 (`.gitignore`에 이미 추가됨)
- 클라이언트 사이드에서 사용하므로 브라우저에 노출되지만, 플랫폼 도메인 제한으로 보호됩니다

### 2. 플랫폼 도메인 등록 필수
- 등록되지 않은 도메인에서는 지도 API가 작동하지 않습니다
- 개발: `http://localhost:5173`
- 배포: 실제 배포 도메인 추가 필요

### 3. 무료 할당량
- 카카오 지도 API는 무료 할당량이 있습니다
- 일일 300,000건 / 월 10,000,000건
- 초과 시 과금될 수 있으니 사용량 모니터링 필요

## 🔧 트러블슈팅

### 문제 1: 지도가 표시되지 않음
**원인:**
- App Key가 잘못 입력됨
- 플랫폼 도메인이 등록되지 않음

**해결:**
1. `.env` 파일의 `VITE_KAKAO_MAP_APP_KEY` 확인
2. Kakao Developers에서 플랫폼에 `http://localhost:5173` 등록 확인
3. 개발 서버 재시작

### 문제 2: "지도를 불러오는 중..." 메시지가 계속 표시됨
**원인:**
- SDK 로드 실패

**해결:**
1. 브라우저 개발자 도구(F12) → Console 탭 확인
2. 네트워크 오류 확인
3. App Key가 정확한지 재확인

### 문제 3: CORS 오류
**원인:**
- 플랫폼 도메인 미등록

**해결:**
- Kakao Developers → 플랫폼 → Web 플랫폼에 현재 도메인 추가
